# Font for UAE number plate generator (optional)

For the correct plate look, use **GL Nummernschild** (German license plate font) or **DIN 1451 Mittelschrift**:

**Preferred – GL Nummernschild**
- **Filenames (any one):** `GL-Nummernschild.ttf`, `GL-Nummernschild.woff`, `GL-Nummernschild.woff2`, or variants like `GL-Nummernschild-Mtl.ttf` / `GL-Nummernschild-Eng.ttf`
- **Source (free):** [FontSpace](https://www.fontspace.com/gl-nummernschild-font-f12469), [1001 Free Fonts](https://www.1001freefonts.com/gl-nummernschild.font)

**Alternative – DIN 1451 Mittelschrift**
- **Filenames:** `DIN1451Mittelschrift.woff2`, `.woff`, or `.ttf`
- **Source (free):** e.g. [Peter Wiegel – Alte DIN 1451](http://www.peter-wiegel.de/alteDin1451.html)

The generator tries GL Nummernschild first, then DIN 1451, then **Barlow Bold** from Google Fonts if no font file is found.
